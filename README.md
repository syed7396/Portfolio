# Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5, CSS3, and Vanilla JavaScript**. This project showcases my skills, projects, and contact information with a clean user interface and interactive animations.

---

## Features

- Responsive design for mobile, tablet, and desktop
- Modern hero section
- About section
- Skills section
- Projects showcase
- Contact form
- Smooth scrolling
- Scroll animations
- Scroll-to-top button
- Professional dark theme

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
